# NestJs example app

Following NestJs [starting guide](https://docs.nestjs.com/first-steps)
```bash
$ npm i -g @nestjs/cli
$ nest new nestjs-example
```

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:debug
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
